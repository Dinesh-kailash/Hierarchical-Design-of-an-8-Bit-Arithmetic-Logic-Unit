# Hierarchical-Design-of-an-8-Bit-Arithmetic-Logic-Unit
Designed and implemented an 8-bit Arithmetic Logic Unit (ALU) using Digital Logic and Verilog HDL. The project features a hierarchical architecture consisting of a 1-bit Full Adder, 8-bit Ripple Carry Adder, Add/Subtractor, Logic Unit, and Shift Unit, supporting multiple arithmetic and logical operations.
